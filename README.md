## WEB-TEST-PRODEQUA

- VueJS 3 (CDN)
- TailwindCSS 2

### INSTALL DEPENDENCIES AND START SERVER
Install dependencies declared at package.json file
- npm install

Compile Tailwindcss styles
- npm run build

Run server
- live-server public
